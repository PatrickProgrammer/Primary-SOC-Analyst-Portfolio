# Security policy

This repository is a sanitized portfolio project.

## What is not included

- API keys
- Credentials
- Private endpoint data
- Client data
- Production logs
- Internal URLs
- Institutional identifiers
- Private source material

## Handling sensitive data

All screenshots and samples should be reviewed before publication. Any names, IDs, emails, dates, ticket keys, internal hostnames, or organization-specific details should be removed or replaced with generic values.

## Responsible use

This project is intended for defensive security learning and portfolio demonstration only.
